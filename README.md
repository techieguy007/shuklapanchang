# Shukla Panchang (React)

Minimal Vite + React scaffold for a Panchang homepage.

Getting started

1. Install dependencies:

```powershell
npm install
```

2. Run development server:

```powershell
npm run dev
```

Build:

```powershell
npm run build
npm run preview
```

- [src/main.jsx](src/main.jsx)
